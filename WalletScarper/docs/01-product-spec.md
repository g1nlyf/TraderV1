# WalletScarper Final Free v1

Цель системы - 24/7 заменить ручной процесс поиска хороших Solana-кошельков.

Пользователь раньше вручную открывал GMGN, DexScreener и Solscan-подобные интерфейсы, находил свежие, но уже живые токены, смотрел трейдеров, выбирал тех, кто закрыл токен в плюс, и проверял их долгосрочную историю. Final Free v1 делает эту воронку автоматически.

Система не торгует реальными деньгами. В ней нет приватных ключей, реальных ордеров и отправки транзакций. В версии v1 есть только сбор данных, scoring, tracked wallets, Telegram, dashboard и paper trading log.

Главная метрика - `copyability_score`, а не PnL. Хороший кошелек должен быть прибыльным, достаточно медленным для копирования, не похожим на sniper/bundler/high-frequency bot, иметь нормальный размер сделок и не зависеть от одного случайного x50.

## Что делает система

1. Ищет свежие Solana pools через бесплатные источники.
2. Фильтрует токены по возрасту, ликвидности, объему, активности и buy/sell balance.
3. Собирает сделки по лучшим pools.
4. Параллельно слушает live DEX trades через Bitquery CoreCast, если бесплатный доступ работает.
5. Группирует сделки по `(wallet, token)`.
6. Считает realized PnL через FIFO.
7. Агрегирует wallet-level метрики.
8. Считает `bot_score`, `human_score`, `copyability_score`.
9. Добавляет лучшие кошельки в `tracked_wallets`.
10. Показывает рейтинг в Telegram и web dashboard.
11. Отдельно, если включено, отправляет live alerts по tracked wallet покупкам.

## Бесплатность

Core pipeline не требует платных API. Система использует free tiers и public endpoints. Если источник получает rate limit, система должна снижать глубину анализа, ждать reset и продолжать работу без обхода лимитов через multiaccount/key rotation.

GMGN используется как output link, а не core source.
